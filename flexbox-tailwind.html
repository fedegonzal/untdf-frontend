Flexbox Layout - UNTDF

Este es un ejemplo de diferentes layouts utilizando Flexbox, desarrollado por la Universidad Nacional de Tierra del Fuego para los alumnos de las carreras de Informática y Sistemas. Los layouts presentados son responsive y se recomienda navegarlos con el inspector del browser emulando diferentes dispositivos: teléfonos, tablets, notebooks y escritorio.
